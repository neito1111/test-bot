## BotPanel — Telegram bot для Дроп‑менеджеров / Разработчика / Тим‑лида

### Что сделано
- **Дроп‑менеджер**: запрос доступа → смена (начать/закончить) → создание анкеты → правка отклонённой анкеты
- **Разработчик**: апрув/отклонение запроса доступа
- **Тим‑лид**: лайв‑анкеты (апрув/отклонить с комментом) + условия по банкам (скрины/кол-во/инструкция)
- **Группа**: подтверждённые анкеты автоматически отправляются в чат

### Требования
- Python **3.11+** (проверено на **3.13**)

### Установка
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp env.example .env
```

### Настройка
Отредактируй `.env`:
- `BOT_TOKEN`
- `DEVELOPER_IDS` (id разработчиков через запятую)
- `TEAM_LEAD_IDS` (id тим‑лидов через запятую)
- `GROUP_CHAT_ID` (id группы для оповещений)

### Запуск
```bash
python -m bot
```

### Добавление бота в группу
- Добавь бота в группу и дай право **отправлять сообщения**
- Узнай `GROUP_CHAT_ID` (например через @RawDataBot) и впиши в `.env`

### Примечания по форвардам
Telegram не всегда даёт номер телефона из пересланного сообщения. Бот сохраняет максимум доступного: `id`, `username`, `first_name`, `last_name`, текст и дату форварда.
